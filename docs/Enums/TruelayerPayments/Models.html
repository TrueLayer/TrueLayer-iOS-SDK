  • PaymentId

Payment Token

  • PaymentToken

    It identifies a Token for a Payment.

    The token is obtained by calling the POST /payments endpoint to create a new payment. The token contains information about the scope of the payment, and other data and it is used as a security check that the payment is legit. Check the doc at this url for further information about how to create a payment.
