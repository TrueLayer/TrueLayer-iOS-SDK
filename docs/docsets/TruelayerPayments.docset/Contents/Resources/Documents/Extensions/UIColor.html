  • init(lightApperance:darkApperance:)

    Creates UIColor that supports both LightApperance and DarkApperance.

    Use if you want different colors for different modes. For pre-iOS 13.0 & tvOS 13.0 os versions this will return the UIColor with lightApperance value.
