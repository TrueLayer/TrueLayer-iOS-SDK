  • UIStyle

    Controls TrueLayer UI SDK Styling. Please use to change various styling options on the SDK.

    Truelayer UI SDK comes with defaults defined on the init signature. Please see UIStyle value for more information.

    It’s possible to customise individual properties when initializing the object change how the SDK appears . For example:

       let viewStyle: UIStyle = UIStyle(primary: .red)

    The Truelayer SDK supports dark mode. It exports a convenience init to create a UIColor that support both the light and dark appearances.

       let blueOnBlack = UIColor(lightAppearance: UIColor.systemBlue,
           darkAppearance: UIColor.black)

    If you specify a single color, as in the first example above, then that color is used for all the appearances. The SDK works with colors loaded from a resource catalog as well.
