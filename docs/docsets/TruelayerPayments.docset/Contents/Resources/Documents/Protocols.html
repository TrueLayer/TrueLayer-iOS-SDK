  • TruelayerPaymentsManager

    It describes the interface of the TruelayerPayment Manager APIs

    We suggest to grab a concrete implementation of this protocol by using the .shared static property of the TruelayerPayments.Manager type. Then invoke the start method as soon as possible. The best place to do this is in the AppDelegate, SceneDelegate, or in the object responsible to initialize all the dependencies of your app.
