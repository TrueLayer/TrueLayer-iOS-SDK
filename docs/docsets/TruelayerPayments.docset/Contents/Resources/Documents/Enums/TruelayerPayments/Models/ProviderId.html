ProviderId

public struct ProviderId : RawRepresentable, Hashable, ExpressibleByStringLiteral

It identifies a Provider.

The id is returned by the SDK when the user chooses a provider so that the app can create a Payment.

Warning

This is required by the v2 of the Payment API. We could remove this when moving to the v3.
